📈 Google Analytics 4 (GA4) 集成说明

项目 说明
官网 https://analytics.google.com
价格 免费（基础版够用）
注册要求 Google账号 + 网站域名
数据延迟 通常 24-48 小时

✅ 如何获取你的 GA4 测量ID

  1. 访问 Google Analytics 并登录
  2. 点击左下角"管理" ⚙️
  3. 在"资源"列，点击"数据流"
  4. 点击你的网站数据流
  5. 找到"测量ID"，格式类似 G-XXXXXXXXXX

📋 GA4 代码示例

<!-- 1. 在 <head> 中添加全局代码 --> <script async src="https://www.googletagmanager.com/gtag/js?id=G-YOUR_ID"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){ dataLayer.push(arguments); }; gtag('js', new Date()); gtag('config', 'G-YOUR_ID'); </script> <!-- 2. 追踪页面浏览（含UTM参数） --> <script> function getUrlParam(name) { var params = new URLSearchParams(window.location.search); return params.get(name) || '(not set)'; } gtag('config', 'G-YOUR_ID', { 'page_title': '测试页面', 'page_location': window.location.href, 'utm_source': getUrlParam('utm_source'), 'utm_medium': getUrlParam('utm_medium'), 'utm_campaign': getUrlParam('utm_campaign') }); </script>

📊 GA4 能看到的数据

  • 实时数据：当前在线用户、正在访问的页面
  • 用户来源：按 utm_source/medium/campaign 分组
  • 转化追踪：注册、购买、表单提交等目标
  • 用户行为：页面停留、跳出率、滚动深度

📉 百度统计 集成说明

项目 说明
官网 https://tongji.baidu.com
价格 免费
注册要求 百度账号 + 网站域名（需备案）
数据延迟 通常几小时，实时功能需付费

✅ 如何获取你的百度统计代码

  1. 访问 百度统计 并登录
  2. 点击"管理" → "新增网站"
  3. 填写网站信息（域名需备案）
  4. 复制生成的 hm.js?xxxxxxxx 代码

📋 百度统计代码示例

<!-- 百度统计代码（放在 <body> 最前面） --> <script> var _hmt = _hmt || []; (function() { var hm = document.createElement("script"); hm.src = "https://hm.baidu.com/hm.js?YOUR_ID"; var s = document.getElementsByTagName("script")[0]; s.parentNode.insertBefore(hm, s); })(); </script> <!-- 手动追踪UTM来源（可选） --> <script> if (_hmt) { _hmt.push(['_setCustomVar', 'utm_source', getUrlParam('utm_source')]); _hmt.push(['_setCustomVar', 'utm_medium', getUrlParam('utm_medium')]); } </script>

📊 百度统计能看到的数据

  • 流量分析：PV/UV、来源地域、时段分布
  • 来源分析：搜索引擎、入口页面、搜索词
  • 受访页面：热门页面、页面停留、跳出率
  • 转化分析：设置转化目标后追踪

⚖️ GA4 vs 百度统计 对比

对比项 Google Analytics 4 百度统计
网站备案 ❌ 不需要 ✅ 需要（国内要求）
国内数据 ⚠️ 准确度一般 ✅ 更准确（基于百度搜索）
国际数据 ✅ 全球覆盖 ❌ 仅限国内
功能深度 ✅ 更强大、灵活 ⚠️ 基础功能够用
学习成本 ⚠️ 英文界面，稍复杂 ✅ 中文界面，易上手
推荐场景 外贸/跨境/多语言网站 纯国内业务网站

💡 建议：两者同时安装

对于国内业务网站，建议同时安装 GA4 + 百度统计，取长补短：
• 百度统计：看国内搜索流量（百度）
• GA4：看多渠道综合数据、用户行为分析

📝 如何在真实环境中验证

  1. 安装好 GA4/百度统计后，清除浏览器缓存
  2. 用带UTM参数的链接访问你的网站
  3. 等待 24-48 小时（GA4）或几小时（百度）
  4. 登录分析工具后台，查看"用户获取"或"来源/媒体"报告
  5. 你应该能看到 utm_source = ai 的数据
